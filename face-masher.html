Face Masher

Synopsis

Face Masher is a casual game in which you try to put faces together while their parts (eyes, nose, mouth, etc) are falling down.

Videos

About The Project

The initial idea behind this project was to create a very simple but interesting gameplay that felt very casual and if we were happy with it, reach out to publishers.

The first prototype had a mechanic pretty similar to the one in Face Masher (except it was more of a building game) and it got the attention of a very successful publisher. They gave us the green light to further develop the game so we could have a testable prototype ready for marketing.

The marketing tests came back with interesting results, but they weren't good enough to dedicate more time and resources to this project, so we put it aside.
