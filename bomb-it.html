About The Project

Our idea for Bomb It was to make a game with a core mechanic that required very simple input while also allowing for some iteration.

After creating a prototype, we felt that it had lots of potential so we reached out to publishers and as we expected, we quickly got a deal.

The publisher gave us the green light to further develop the project and so we did, launching the game in the USA and Canada for market testing and CPI analytics. The results weren't as optimal as we wished so we decided to stop allocating time and resources on this project.
