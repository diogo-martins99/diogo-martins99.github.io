About The Project

On the final week of 2020, my college teammates and I decided to participate in the Weekly Game Jam 181 whose theme was "New Year's Countdown".

The four of us hadn't been involved in game jams in a long time but we always like to challenge ourselves so we decided to make our first 3D jam and, even more ambitiously, a multiplayer online game!

At the time, I was working at Nerd Monkeys for a few months so my programming skills had already improved substantially compared to when I was in college. Something that this project definitely reflected.

It's worth stating that this game was developed in only 6 days, which is quite an accomplishment from our perspective!

My Role In The Project

  • Gameplay Programming

    I was the gameplay programmer, implementing everything from scratch.

  • Online Programming

    Rui was the online programmer and did almost all of the online implementation. But I also got the chance to work on it after I was finished with the gameplay programming.
    The online implementation was done using the Photon and SmoothSync assets.

  • Game Design

    I've worked on the game's concept and game design.
