Synopsis

Planet Free is a contemporary treasure hunt. A Pokemon GO like game that connects players to their favorite brands by rewarding them with vouchers, discounts, tickets, prizes, and exclusive products.\

About The Project

Planet Free is a work-for-hire project which we developed at Fun Punch Games for a specific client. The production was somewhere at the one-year mark when I joined to help accelerate and improve its development. I was mostly responsible for augmenting the gamification and revamping the aesthetic.
The game was officially launched in August 2023.

My Role In The Project

  • Gameplay Programming

    I implemented gameplay systems, mechanics and specific tools that helped the development of the project.

  • UI Systems Programming

    I was responsible for the implementation of many UI systems.

  • Art Pass

    I was in charge of revamping and embellishing many of the games visuals, more specifically the world, the rescue mini-game and some UI (you can check some of the work samples here).

  • All-Round Juice

    I was responsible for sprucing up visuals and improve game feel by tweaking animations and adding VFX along with custom shaders.
